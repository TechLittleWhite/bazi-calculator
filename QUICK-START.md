# 🚀 快速开始指南

**下次打开项目时，按以下步骤操作即可快速恢复工作！**

---

## ⚡ **30秒快速启动**

### **1. 打开终端，进入项目**
```bash
cd /Users/x/bazi-calculator
```

### **2. 启动开发服务器**
```bash
# 启动后端（终端1）
cd backend && node server.js &

# 启动前端（终端2）
cd frontend && npm run dev
```

### **3. 开始工作**
- 浏览器访问：http://localhost:3000
- 开始开发或测试
